# 空手道部のサイト
空手の動作を3Dで見られるサイトです（未完成）

サイトはNuxt,Vuejs、3DモデルはVIBE(https://github.com/mkocabas/VIBE)を用いて作成しています。

電気通信大学(http://www.karatedo.club.uec.ac.jp/)と横浜市立大学(https://ycu-karate.jimdosite.com/)の空手道部での共同開発です。
